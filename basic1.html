HTML stands for Hyper Text Markup Language. It is the standard markup language for creating Web pages and it describes the structure of a Web page consisting of a series of elements. These elements tell the browser how to display the content.
Hypertext refers to the way in which Web pages are linked together and it is markup language which is tags tell the browser how the page will be rendered on it.
